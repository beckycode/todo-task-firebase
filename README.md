# todo-task-firebase

This project is a sandbox to practice with CRUDs in Firebase
It is a small project built with:

- HTML
- Javascript
- [Boostwatch](https://bootswatch.com/) - morph theme

> [!NOTE]
> Bootswatch is a page where you can find themes for Bootstrap
